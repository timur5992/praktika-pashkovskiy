## 1. Описание проекта

**«Тепломастер Плюс»** — это WPF-приложение для автоматизации учёта ремонтов теплового оборудования.  

Программа позволяет управлять:
- заявками на ремонт
- клиентами
- сотрудниками
- складом запчастей

## 2. Технологический стек

- **Язык программирования:** C# (WPF, .NET Framework)  
- **Система контроля версий:** Git / GitHub  
- **Среда разработки:** Visual Studio 2022  

## 3. Структура проекта

- **pashkovskiy** — исходный код приложения (логика, разметка XAML)  
- **pashkovskiy.sln** — файл решения Visual Studio  
- **README.md** — документация проекта  

## 4. Результаты практики

### 4.1. Работа с Git

1. **Инициализация:** Создан локальный репозиторий, выполнена привязка к удаленному репозиторию на GitHub.  
2. **Ветвление:** Создана ветка `feature-ui` для разработки интерфейса, что позволило изолировать процесс разработки от стабильной версии в `main`.  
3. **Слияние:** После проверки кода выполнено слияние ветки `feature-ui` в основную ветку `main`.

### 4.2. Проведение Code Review

Был проведен анализ фрагмента кода `MainWindow.xaml.cs`.

- **Объект анализа:** Логика управления кнопкой возврата (`BtnBack`).  
- **Результат:** Замена `Visibility.Hidden` на `Visibility.Collapsed`.  

Это позволило динамически перестраивать интерфейс и исключить пустые зоны в макете.

### 4.3. Обратное проектирование алгоритма

**Алгоритм навигации:**

1. Пользователь нажимает кнопку меню  
2. `MainFrame` переходит на страницу  
3. Срабатывает `ContentRendered`  
4. Если страница `DashboardPage` → кнопка "Назад" скрывается  
5. Если другая страница → кнопка "Назад" показывается  
6. При нажатии "Назад" → возврат на предыдущую страницу  

## 5. Запуск

Для запуска приложения необходимо открыть файл **`pashkovskiy.sln`** в среде Visual Studio и выполнить компиляцию проекта **(F5)**.
