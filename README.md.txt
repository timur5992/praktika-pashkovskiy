1. Описание проекта
"Тепломастер Плюс" — это WPF-приложение для автоматизации учёта ремонтов теплового оборудования.
Программа позволяет управлять заявками, клиентами, сотрудниками и складом запчастей.

2. Технологический стек
Язык программирования: C# (WPF, .NET Framework)
Система контроля версий: Git / GitHub
Среда разработки: Visual Studio 2026

3. Структура проекта
pashkovskiy/ — исходный код приложения (логика, разметка XAML).
pashkovskiy.sln — файл решения Visual Studio.
pashkovskiy.md — файл описания проекта (документация).


4. Результаты практики

4.1. Работа с Git 
Инициализация: Создан локальный репозиторий, выполнена привязка к удаленному репозиторию на GitHub.
Ветвление: Создана ветка feature-ui для разработки интерфейса, что позволило изолировать процесс разработки от стабильной версии в main.
Слияние: После проверки кода выполнено слияние ветки feature-ui в основную ветку main.
4.2. Проведение Code Review**
Объект анализа: Логика управления кнопкой возврата (BtnBack) в MainWindow.xaml.cs.
Результат: Замена Visibility.Hidden на Visibility.Collapsed. Это позволило динамически перестраивать интерфейс и исключить пустые зоны в макете.
4.3. Обратное проектирование алгоритма**
Алгоритм навигации:
1. Пользователь нажимает кнопку меню
2. MainFrame переходит на страницу
3. Срабатывает ContentRendered
4. Если страница DashboardPage → кнопка "Назад" скрывается
5. Если другая страница → кнопка "Назад" показывается
6. При нажатии "Назад" → возврат на предыдущую страницу

5. Запуск
Для запуска приложения необходимо открыть файл pashkovskiy.sln в среде Visual Studio и выполнить компиляцию проекта (F5).
